::: tip

解释：在过去的某个时间点上，动作已完成了”

:::


::: tip

构成：助动词have变位【过去式（had）】 + 动词的过去分词

**<font color="#e38787">时间部分：have变位（had） -  表示时间（过去）</font>**

**<font color="#6f9d40">状态部分：动词的过去分词（ed） - 表示状态（完成）</font>**

:::
