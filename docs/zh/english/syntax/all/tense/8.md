::: tip

解释：时间为“过去”，状态为“完成一部分 并且 还在进行。在过去的某个时间点，动作已经完成了一部分，但还要继续做”

:::


::: tip

构成：have变位【过去式（had）】 +   been   + 现在分词

:::
