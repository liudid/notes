::: tip

解释：时间为“现在”，状态为“完成一部分 并且 还在进行”

:::


::: tip

- 现在完成时：have + 过去分词

- 现在进行时：be + 现在分词

构成：have变位 +   been   + 现在分词

**<font color="#e38787">时间部分：就是将 “现在完成时” 中的 have变位【现在（时间）】拿出来</font>**

**<font color="#6f9d40">状态部分：在将 “现在进行时” 中的 be变位【现在（时间）】拿出来，并标记为完成，既为been。在加上现在分词</font>**

:::
