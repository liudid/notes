::: tip

解释：时间为“现在”，状态为“完成”

:::


::: tip

构成：“助动词have变位”（have/has）+ 动词的“过去分词”

**<font color="#e38787">时间部分：have变位（have/has） -  表示时间（现在）</font>**

**<font color="#6f9d40">状态部分：动词的过去分词（ed） - 表示状态（完成）</font>**

:::
